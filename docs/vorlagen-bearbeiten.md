# Vorlagen bearbeiten

## Übersicht

Im Tab **Vorlagen bearbeiten** passen Sie Layout-Vorlagen per KI an – mit Vorschau und Bestätigung vor dem Speichern.

## Vorlagen-Typen

- **System-Vorlagen** (gelb/blau): nur per **Klonen** veränderbar
- **Eigene Vorlagen**: bearbeitbar, löschbar, importierbar

## Workflow

### 1. Vorlage wählen oder klonen

- Klicken Sie eine Vorlage in der Galerie
- **Klonen** über Menü *Vorlagen → Klonen* oder Kontext in der Galerie

### 2. Änderungswunsch eingeben

Beschreiben Sie grob, was sich ändern soll, z. B.:

- „Neon-Rahmen von blau auf rot"
- „Logo im Header größer"
- „Kontaktleiste kompakter"

### 3. Prompt optimieren

Die KI analysiert die Vorlage und erstellt einen präzisen Bearbeitungs-Prompt. Sie können ihn vor dem Anwenden anpassen.

### 4. Änderung anwenden

Die KI erzeugt eine Vorschau. Links: Original, rechts: KI-Ergebnis.

### 5. Entscheidung

- **Änderung akzeptieren** – Vorlage wird gespeichert (System-Vorlagen werden als neue Benutzer-Vorlage angelegt)
- **Änderung verwerfen** – Original bleibt unverändert

## Versionen

Bei Bearbeitung eigener Vorlagen wird die alte Version im History-Ordner archiviert.

## Import

*Vorlagen → Importieren…* – beliebige PNG-Vorlage als Ausgangsbasis.
