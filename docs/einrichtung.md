# Erste Schritte

## 1. WerbungMaker starten

Nach der Installation starten Sie **WerbungMaker** aus dem Startmenü oder der portablen EXE.

## 2. Codex Local Bridge

WerbungMaker prüft automatisch, ob die Bridge läuft (`http://127.0.0.1:8765`).

Falls nicht:

1. WerbungMaker lädt die neueste Bridge-Version herunter (beim ersten Mal)
2. Die Bridge wird gestartet (Tray-Icon in der Taskleiste)
3. **Pairing-Code** aus dem Bridge-Tray-Menü (6 Ziffern) in WerbungMaker eingeben
4. Auf **Verbinden** klicken

## 3. Codex CLI

Die Bridge benötigt die **Codex CLI** mit aktivem Login:

1. Falls nicht installiert, versucht WerbungMaker die Installation über winget/npm
2. Klicken Sie auf **Codex anmelden** oder führen Sie in einem Terminal aus:

```powershell
codex login
```

3. Melden Sie sich im Browser an

## 4. Erste Werbung

1. Tab **Werbung erstellen**
2. Vorlage wählen
3. Referenz-Produktbilder hinzufügen
4. **Prompt erstellen** → **Werbung generieren**
5. Vorschau prüfen und **Als PNG speichern**

## Hilfe in der App

Menü **Hilfe** oder Tab **Hilfe** in der App öffnen.
